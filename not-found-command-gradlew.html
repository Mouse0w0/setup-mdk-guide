# 找不到命令 gradlew

在PowerShell运行gradlew命令时，提示找不到命令 gradlew，但它确实存在于当前位置。

查看图片

# 解决方案

将gradlew命令改为.\gradlew命令，例如：

.\gradlew setupDecompWorkspace
1
其他解决方案

在当前目录下打开CMD命令提示符，并运行原命令。

gradlew setupDecompWorkspace
1
